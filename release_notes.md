# Release Notes - Version 1.11.0

New Features:
- Excel File Handling: Added the excel package to handle the export and import of financial data in Excel format.
- Financial Report Export: Implemented functionality to export financial reports as Excel files, making it easier to share and analyze data offline.

Enhancements:
- LaporanKeuanganModel: Enhanced the LaporanKeuanganModel to support date filtering for fetching financial records, allowing for more flexible and precise data retrieval.
- UI Improvements: Updated various UI components to handle optional fields and improved data presentation across multiple screens, resulting in a cleaner and more intuitive user experience.

Dependency Updates:
- Updated Dependencies: Updated the versions of dependencies in pubspec.lock. Notably, the versions of archive and flutter_native_splash were downgraded to ensure compatibility.
- General Updates: Other minor updates to ensure better compatibility and stability.
